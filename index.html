El proyecto consiste en el desarrollo de un sistema de matrícula estudiantil utilizando phpMyAdmin, XAMPP, MySQL, C# y Guna Framework, que facilita la gestión administrativa de una institución educativa. El sistema de ventas desarrollado en C# .NET Framework, Guna Framework y SQL Server permite a los usuarios iniciar sesión con roles diferenciados (administrador o empleado), ofreciendo funcionalidades específicas para cada uno. Los administradores tienen acceso completo al CRUD de usuarios, mantenimiento de categorías, productos y datos para facturación, registro y visualización detallada de ventas y compras, gestión de clientes y proveedores, y generación de informes. Por otro lado, los empleados pueden registrar ventas y compras, gestionar clientes y proveedores, así como generar informes limitados. La interfaz del sistema es intuitiva y eficiente, facilitando las operaciones diarias de venta y gestión empresarial.El empleado, dentro del sistema de ventas, tiene la capacidad de registrar ventas y compras, así como gestionar clientes y proveedores. Además, puede generar informes limitados sobre las ventas y compras realizadas.
